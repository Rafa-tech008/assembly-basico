# Assembly MIPS - Básico

## 📖 Objetivo

Reunir exercícios acadêmicos desenvolvidos em Assembly MIPS com foco nos conceitos fundamentais da linguagem e da arquitetura de computadores.

## 📂 Conteúdo

O repositório contém exercícios sobre:

- Registradores;
- Operações aritméticas;
- Entrada e saída de dados;
- Estruturas básicas de programação.

## 💻 Tecnologias

- Assembly MIPS
- MARS Simulator

## ▶️ Como executar

1. Abra o arquivo no MARS Simulator.
2. Monte o programa (Assemble).
3. Execute o código (Run).

## 🎓 Disciplina

Arquitetura e Organização de Computadores – Fatec Zona Leste
